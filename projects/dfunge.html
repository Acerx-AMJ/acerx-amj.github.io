About

2D esoteric programming language based off Befunge-93 with file I/O, functions and other QOL elements. Made with C++.

Features

The playground is laid out on an infinite 2D grid. The program counter keeps track of the position and velocity which has inertia just like in Befunge-93, meaning it will continue moving in the same direction until it is changed. When the program counter lands on a command, it gets executed. All commands and functions can be found on Github.

Dfunge uses four data structures - the stack, the defered commands, variables and registers. Each have their pros and cons. This probably tells you nothing about the language, so here are some examples. The classic "Hello, World!" can be written like this:

o"Hello, World!"t,E

Or like in the thumbnail:

v%mod   vQ,-*  3i  t*  H+  2t   DIQxo  X,,\<
'2  >,  x6     -H  ,8  -H, 3/  ,t      ,E
7#  HX  >",",  sq  QS  '32+1*  >Q HH,  +vxEH
2a  ^+  3,     Hi  +t  *t t,+  ++  3t  t<
>,tt*   i,      r>ro   qr  t2   %pow   42*+^

This program loops from 1 to 100 and outputs all numbers to the terminal:

> 0 #i >   '100 @i Gj E
       ^,t.i@ i#i i@<

This is my implementation of factorial. It takes a number from the user and prints out its factorial.

`H #n #r>          @n 1Gj @r. t, E
        ^r#*n@r@ n# d n@<

The all favorite guessing game. It uses labels and built-in functions. You can play it infinitely.

:main  '100 %randint #num ;guess
:guess o"Guess the number 0-100: "` #guess ;eval
:eval  @guess @num =j @guess @num Gj ;toolow
                    > ;correct     > ;toohigh

:correct o"You guessed correctly!"t, o"Play again [y/n]: "~H,t, "y"=jE
:toolow  o"Too low!"t,  ;guess                                      >;main
:toohigh o"Too high!"t, ;guess

This is a four command calculator that uses deferred commands. It loops continuously until the user decides to exit it.

;inputnum #num1 ;inputnum #num2 ;inputop #op ;handleop E

:inputnum o"Input a number: "` R
:inputop  o"Input the command (+, -, *, /): " ~H,t, R

:handleop @op "+"H=jq @op "-"H=jq @op "*"H=jq "/"H=jq o"Invalid command."t, R
                   >           >           >       > ;getresult R

:getresult HT @num1. o" ",o" " @num2. o" = " @num1 @num2 x. t, R

Dfunge can also do file I/O. This example shows how to iterate a directory. The program simply outputs all files in the given directory.

o"Input a directory: "&s %iteratedirectory > #size 0#i>@size @i =!j t, H0=!j E
                                                      ^    i#i i@,<
                                           ^                               <

Source code for these examples and even more examples can be found on Github.

Development

In August 2025 I made Cfunge - a language based off Befunge-93. It was really similar to Befunge-93 but it was more focused on functions and had some modern features. Later - in December 2025 - I recreated it and called it Dfunge. It had more functions, was cleaner and easier to use. You can compare the calculator project in the two languages I made:

Cfunge

>o"Input command (+, -, /, *, q): "~C@ C,t, C"q"=;e_ 01J E@ 02J E;R_o"Invalid command."t,R
>o"Input first number: "`Y@ o"Input second number: "`Z@R
>$+$03J $-$03J $/$03J $*$03JR
>CPO=!;R ;Q_ Y.PO,Z.o"="YZx. 1E@ t,R

Dfunge

;inputnum #num1 ;inputnum #num2 ;inputop #op ;handleop E

:inputnum o"Input a number: "` R
:inputop  o"Input the command (+, -, *, /): " ~H,t, R

:handleop @op "+"H=jq @op "-"H=jq @op "*"H=jq "/"H=jq o"Invalid command."t, R
                   >           >           >       > ;getresult R

:getresult HT @num1. o" ",o" " @num2. o" = " @num1 @num2 x. t, R

The reason I even made Cfunge was that I started implementing super tiny esoteric languages and that was so fun that I decided to move to Befunge-93. It's a hard language to implement at first but once you get it it's quite simple. But I wasn't happy with just the language implemented, I had to make it my own thing. I loved it because it was so unique.

And like so, feature by feature, I made Cfunge my own language. A few months later the itch to make esoteric languages came back and I decided to remake it again. This time I actually documented it. Here's the source code for Cfunge and Dfunge, as well as a link to Befunge-93's page.
