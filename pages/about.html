About

I'm Daniel - a low-level programmer and game developer. My favorite types of projects are games and I like to work on projects from embedded systems to simple websites. I mostly code with C/C++ and I love what I do.

I primarily work in VS Code on Linux or in Visual Studio on Windows and I know how to use tools like Git, Perf and Valgrind. I'm really flexible and can learn new technologies easily.

Outside of coding I enjoy taking walks and experimenting with electronics. I don't know much yet, but it's super fun to do.

The goal behind this website was to create a personal portfolio and a blog. Since I like to journal, I've been wanting to expand it and write blogs as well. Only I have access to write and edit blogs. They go over my interests, opinions and updates on projects. My blogs are accessible to anyone free of charge or advertisements.
