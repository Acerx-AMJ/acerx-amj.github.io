About

I'm Daniel. I program games and low-level projects in C++. I also like working on embedded systems and simple websites.

I've worked both on Linux and Windows and use VS Code or Neovim as my IDE, and can code with Lua, Java and HTML/CSS/JS as well. I learn new concepts and tech quickly, especially when I'm interested in it. I'm familiar with Raylib, Candle, SFML and nlohmann-json libraries and use tools like git, cmake, heaptrack and hotspot daily.

Outside of coding I enjoy taking walks, reading and writing. I also like tinkering with electronics and fixing laptops.

The goal behind this website was to create a personal portfolio and a blog. Since I like to journal, I've been wanting to expand it and write blogs as well. Only I have access to write and edit blogs. They go over my interests, opinions and updates on projects. My blogs are accessible to anyone free of charge or advertisements.
